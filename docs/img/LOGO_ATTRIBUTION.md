# Logo Attribution

## libspot-rs Logo

The libspot-rs logo (`logo.svg`) is a Rust-themed adaptation of the original libspot logo.

### Original Design
- **Source**: [libspot logo](https://github.com/asiffer/libspot/blob/master/docs/img/logo.svg)
- **Author**: Original libspot project maintainers
- **License**: The original logo is part of the libspot project, which is licensed under LGPL v3

### Rust Adaptation
The Rust-themed version includes:
- Rust programming language color scheme (#ce422b, #a72145, #8e2f42)
- Gradient effect representing the Rust language colors
- Surface rust texture with oxidation colors (#8b4513, #704214, #5d3410)
- SVG masking to ensure texture only appears within the original shape

### Files
- `logo.svg` - The Rust-themed logo for libspot-rs
- `original-logo.svg` - Copy of the original libspot logo for reference
